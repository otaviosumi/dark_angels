# Dark Angels - segurança de eventos
Um trabalho de base de dados

Utilizando Postgresql:
------
Primeiro se instala a BD:
```
sudo apt-get update && sudo apt-get upgrade
```
Depois se cria um usuário:
```
sudo passwd postgres
```
Loga-se no usuário:
```
su - postgres
```
Para sair do prompt:
```
\q
```
